译文：扁平化网站设计的终极指南

Nov 6th, 2014 4:33 pm

注：原文在 http://www.hongkiat.com/blog/flat-design-resources/

亮色(bright color)和奇特排版(exotic typography)的组合，已经为扁平化界面铺平了道路。这一设计理念的很大部分是极简主义(Minimalism)，你可以看到这些趋势在最近几个月变得如此流行。扁平化网站设计是个卓越的创意，因为它完全关注内容(because it focuses solely on the content)。不过，很多设计者仍然不熟悉最新趋势(2013年, 2012年)，不知如何起步。

在本指南中，我将提供一个终极汇总，包括文章、教程、免费图片和基于扁平化设计的网站布局。任何仍正在涉猎这一概念的人，一定会学到些新东西。这些趋势已经盘旋很长时间，但直到最近，我们才看到扁平化设计重大的兴起。

这一趋势的出现有大量正当的理由。与页面内容相结合的扁平化设计，更易阅读、更易交互(Flat design mixed with page content is easier to read and easier to interact with)。

你不必期望在每个项目中采用扁平化设计。就像所有其它设计趋势，当细节连贯时，扁平化网站设计必须被唯一地使用。否则，会感到不和谐，将一个极简策略应用到一个更复杂的网站。但是，当仔细地放置一起时，我发现扁平化用户界面是卓越、简单而且如此不费吹灰之力就吸引访问者，让他们好几年都是回头客。

扁平网站的流行

正如Forbes上这篇文章谈及的，在业界，扁平化网站设计越来越成熟(a growing maturity within the industry)。它剔除了很多过于复杂的纹理(textures)、模式(patterns)、阴影(shadows)、气泡(bubbles)、渐变(gradients)，和其它闪亮的效果(shiny effects)，以便聚焦在简单性上。但是，扁平化的按钮仍然看起来可点击，输入域仍然看起来可编辑。总的来说，扁平化设计的网站仍然具有交互性，但是没有了非常强烈精炼的图形。

扁平化设计已经盘旋了一段时间，但是最近从Windows移动平台，即现在的Window 8，开始加速。Metro Design已经开启了趋势，它将球传给扁平化布局。在web和移动界面都已经看到重要的提升，??在时髦的新的用户界面上。并且随着响应式网站使得桌面和移动设备的差异变得模糊，??扁平化UI用了一点文化层次来弥合了简单性的隔阂(flat UI has bridged the gap for simplicity with a bit of cultural class)。

当设计一个扁平化网站时，你将面临的最大问题是交互性元素。访问者需要知道，页面上哪些区域是朴素的静态内容，而其它区域哪些是可点击的、可拖拽的，哪些不是。扁平化设计仍然使用淡阴影和边缘效果(light shadows and edge effects)来暗示这一交互性。但这个游戏很微妙(subtlety is the name of the game)，你能目睹很多其它扁平化网站布局的例子。

有帮助的设计者指南

有相当多的在线文章，我没有列出多少。但是，我想分享一些特殊的资源，有助于理解扁平化设计的概念。我愿意把它们更多看做指导方针而非实际的规则。

在一个媒体博客的帖子引用了扁平化设计和UI问题(flat design & UI problems)，作者Missy Titus解释扁平化UI必须有助于你解决问题或者仅仅让它们更容易被看到。当剥掉纹理、??结合处(stitching)和盒子阴影，你不能从页面移除重要的内容。设计的趋势仅仅是一种确定的外观。

当然，扁平化设计不是针对所有网站。有如此多种多样的网站趋势，它们都由于某个原因而流行 – 人们喜欢多样性(people like variety)。仔细精巧制作的2-D界面引人注目，因为它们真的看起来使用简单(they look really simple to use)。它使访问者更多聚焦在你的内容而非其它的。另一篇关于扁平化设计的原理(the principles of flat design)，将帮助你理解更多具体的界面特色。

最值得一提的是，你想聚焦在页面里的内容层级(content hierarchy within the page)。元素通常彼此不会交叉，页面必须非常自然地流动。这包括在颜色和盒子内容之间的切换(switching between colors and box content)。把页面看做围绕排版而构建，会更容易些。这些文本的不同方形区域(box)将告诉用户如何与页面交互。

有些按钮可能用到扁平化背景颜色，而其它的按钮可能使用形状(shape)或象形文字(pictograms)。对于你的扁平化网站，网页文本和颜色方案(webpage text and color schemes)是两个极端主要的设计概念。

微软最开始让Metro外观(look)流行起来，它有一个挑选颜色的卓越系统(a great system of picking out colors)。这些是重要的，因为颜色让你的网站引人注目。颜色让按钮、链接和标题看起来，不是那么非黑即白(appear not so black-and-white)。但有时，选择一个有效的颜色方案可能困难。

你需要浏览其它设计的例子，并看看专业人士在过去是如何做的。到扁平化设计灵感画廊 (FlatDesign inspiration gallery)去看看，找找类似网站界面的汇集。

Flattened Design Colors 压平的设计颜色

DesignModo已经发布了一个，基于围绕扁平化用户设计的一个非常巨大的资源汇集。当时，就颜色的话题，我想指出另一篇讨论扁平化设计和颜色趋势 flat design and color trends的文章。这些方案自己的构成，是从以(full hue)显示的一种颜色开始，然后移除较低的hue(hue lower)以及黑/灰的混合(mixing in black/grey)。这个方法将为你新的调色板(new color platte)创建多重阴影(multiple shadows)。

任何最亮、最饱和的颜色处于浅或暗的背景(light or dark background)之中，都会很突出。在DesignModo，他们发布了另一个有趣的帖子，关于retro color schemes，它能完美地关联到扁平化网站布局(relate perfectly into a flat website layout)。但是当构建新项目的时候，这个过程事实上需要一些实践和直觉。

如果你尝试将创造性的精力放在扁平化颜色方案上，在线资源Flat UI Colors是一个非常棒的着手处。

有关文章

除过拷贝和机械式重复本文中不落俗套的内容，我还想分享其它一些和扁平化界面设计相关的启发灵感的博客帖子。我已经阅读了无数这个话题的文章，我的视角是精致讲究的(exquisite)。

花几分钟略读下面列表中的2~3篇文章，你可能会吃惊你学到的东西： * The Flattening of Design * Flat Web Design: Trend Or Revolution? * A Look at Flat Design and Why it’s Significant * What You Need to Know About Flat Design * Flat Pixels: The Battle Between Flat Design And Skeuomorphism

免费资源

经过前面的介绍，有些设计理论搞清楚后，我们将转向讨论其它的扁平化设计资源。如果你知道到哪里找，所有的图片和资源在网络上都能找到。对于新的开发者来说，开源代码是深入新设计趋势的一些最好方式。

但是图片对于理解扁平化用户设计也很关键。网站设计和移动app设计都发展成极大的学习领域。考虑如下这些资源，作为实践你扁平化设计创意的学习工具。

图片

设计者和开发者都有图片的需要。符号、图标、logo和其它图片特征在几乎每个网站布局中都能找到。于是，使用扁平化技术的免费图片(freebie graphics)在Internet各处正变得甚至更加流行。这里是一小部分我个人喜欢的、扁平化领域中的UI工具和图标。

Pixeden 扁平图标集(Flat Icons Set)

因为Pixeden神奇的免费和付费图片收集，我简直爱上它了。恰好最近，我发现了一些Dribble的镜头(a number of Dribbble shots)，为一组扁平化图标集做宣传(advertising a free Flat Design Icons set)。这些图标以4卷发布，每卷都包含不同的图标集。卷1和卷2集合的标签不慎一样，但是你能分辨出这是非常巨大的扁平化图标的收集 – 所有都免费使用在不限量的网站上！

Pixeden: Vol #1 - Vol #2 - Vol #3 - Vol #4

Dribble Mirror: Vol #1 - Vol #2 - Vol #3 - Vol #4

多彩的扁平化图标集 （Colorful Flat Iconset)

Freebiesbug大量PSD文件给设计者。一个独一无二的例子，他们针对用户界面的多彩扁平化图标 flat colorful icons。你可以从他们网站上或直接从Dribble 页面下载这个免费的集合。这个集合包含12个基于流行Internet服务的图标设计(12 icon designs which are based on popular Internet services)。当然，这个收集不一定对每个项目都有用，但是了解到在那儿有充足的扁平化免费品供图片设计者们去把玩，会很愉快。

扁平化Web浏览器图标

有很多理由，你可能需要在站点上包含浏览器图标。设计代理(Design agencies)和SaaS公司(SaaS companies，注：Saas, Software as a Service, 软件即服务)经常需要列出他们的特性和支持的客户端。并且这些扁平化的浏览器图标 flat web browser icons是任何新布局的最佳伙伴。你也可以从原始的Dribble镜头(original Dribble shot)下载由Stefano Tirloni发布的这组图标集。

Flatilicious 图标

Flatilicious 图标是由Lukas Jurik刚好上个月在Dribble上发布的图标新品牌。这个精彩的收集包括48个免费的扁平化图标设计，在一个PSD文件中。你可以在任何web或移动项目中自由使用它们。我特别喜欢这个集合的是，有多种图标可供选择。对于设计者来说，这的确是慢慢进入扁平化UI的创造性杰作。

IconShock 扁平化图标

这个美妙的扁平化图标集合，由iconShock发布，包含超过3600个不同的样例。这个设计的特点是，在很多图标中具有轻微光泽的效果(a slight glossy effect on many of the icons)，这使得它从其他类似图标集中凸显出来。这些图标已经被重新调整尺寸，有针对大、中、小网站元素的3种不同配置。这非常好。这是多么出色的免费可用的扁平化图标收集！

Flat Event Planning UI PSD 扁平事件计划UI PSD

来自这个 this events input freebie 的更深颜色主题，通过使用扁平化设计，的确脱颖而出。这个界面自身是为一个网站构建的，用于处理用户输入时间的日程安排，或者可能让参会者报到。当然你可以利用这个免费品仅去练习。但是，我也在想不同的PSD风格连同颜色主题自身，为学习提供了一个漂亮的框架。这个免费品的所有功劳都归于 Dylan Opet。

Flat Dropdown Menu PSD 扁平下拉式菜单PSD

当你设计网站布局时，为灵感收集创意可能会是艰难的。从图片滑块(image sliders)、导航链接、微缩图和确定的下拉菜单，如果没有例子，都很难做头脑风暴(can be difficult to brainstorm without examples)。这个免费的PSD包含一个精彩的扁平化网站下来菜单。图标易于扩展，你能用CSS3效果复制很多相同的盒子阴影。看看 original Dribbble shot，下载一个拷贝，再看另一个预览。

Freebie Flat Website UI Kit 免费扁平化网站UI工具

如果你正在寻找基于web的用户界面元素，那么不用继续找了。由 Andreea Nicolaescu 发布的闪光的 flat UI PSD freebie，在你为网站编码时，是个非常好的起步点。使用这个PSD工具，不仅为了图片，也有助于颜色选择，并与完整的颜色方案吻合。这个PSD是围绕白色的背景构建的，这在极简主义设计者中最常见。你可以在 original Dribbble shot 上看到全屏预览。

Flat UI Kit PSD #1

伴随闪光的扁平化图标，我也想分享一组用户界面工具。这第一个PSD文件 由 Riki Tanone创建，并在Dribble上免费发布。这是个小文件，但是它包含了一些不落俗套的元素，诸如视频播放器和邮件收件箱。色彩的选择很有才华，它们几乎让我想起各种色彩柔和的阴影(various pastel shades)。

Flat UI Kit PSD #2

第二个UI kit 也由Riki Tanone发布，更多聚焦在博客元素。它正好和另一个PSD文件尺寸相同，包含博客元素，比如订阅盒子、日历、撰写文本区、tag输入框和一些其它干净的素材。颜色方案是一致的，但扁平化按钮/元素看起来几乎一样。如果你想快点起步，我建议下载这些工具，然后花时间去实践。

Flat UI Kit PSD #3

这个备选 Flat UI kit 由 Devin Schulz设计，并在Dribble上免费发布。我真的喜欢这个PSD文件，因为它大部分聚焦在常见的网站元素上。输入域、水平&垂直导航、滑动条、按钮，和其它一些页对象。在几乎每个元素上，颜色方案都遵循灰/绿。这看起来简直极好的。

所有这些免费的UI工具，完美地涵盖了扁平化设计文化(encapsulate the flat design culture perfectly)。我相信在线上，还能找到更多更多这样例子。事实上，仅仅通过Dribble搜索搜索flat UI kits，就出现了长长的列表。我鼓励所有的设计者去翻阅类似的例子，看看你能从这些扁平化web页元素中构造出什么。

Flatlyfe UI Kit PSD

设计者Dylan Opet，他创建了上述上面免费的event planning UI，也发布了一个更新的UI Kit。这些元素基于常见的网站对象，比如滑动条、按钮、开关、tag和输入域。我明确地推荐 下载一个copy，在一些实际网站中使用这些设计。甚至基于这些PSD编写CSS3风格代码的过程，也是很大的学习体验。

WDD Flat UI Kit PSD

最近在Webdesigner Depot上有个巨大的帖子，发布了一个新的扁平化UI kit。这个集合设计为更靠近metro趋势，它有一个类似的调色板，界面元素，滑动条，条栏(bar)，其它不落俗套的页面对象。我想这一UI kit也可用于移动app设计。当你输入你的mail地址以注册到它们的新闻邮件之后，就可以自由下载。

Open Source Code 开源代码

前端web开发者总是对最近的趋势感兴趣。Web设计和开发常常在很多关键地方重叠，当你正在为网站的界面开发时，这会经常发生。看看这些免费的代码示例，你可以用在你的扁平化设计项目中。

Flat UI

对web开发者来说，Flat UI kit可能是进入这个趋势的最好资源。DesignModo发布了一个免费版本和一个专业版本，都包含PSD文件和HTML/CSS元素。但是，免费版真的让人震惊，它提供了所有你会需要的主要元素，比如属于域、开关按钮、标题栏、图标等很多。这个免费版也寄放在Github, 以方便那些可能想fork项目并进一步扩展功能的开发者。

iCheck

jQuery iCheck plugin是一个增强你输入按钮的工具。值得注意的是，checkbox和radiobox是两个难以用CSS风格化的元素。iCheck完全免费和开源，对于新开发者，是极好的。你可以通过仅仅一点定制化，构建能完美融入你网站布局的优秀界面。

Flat Radio Buttons

发表在Design & Code It上的免费资源可以用于任何web站点项目。我直到最近才听说过这个站点，当我发现他们的flat radio buttons，看起来简直精美极了。原始的设计基于带有橙绿色radio circles的较暗背景。当在新选择项之间切换时，还有一个小动画。

Boxy Buttons 盒子型按钮

Web开发者可能熟悉基于云的IDE，比如jsFiddle和JSBin。而Codepen是另一个例子，它实际上在一个地方编目和陈列最新的pen。开发者可以练习他们的技能，向世界发布极好的免费代码。

这个展示了以三种不同尺寸设计的flat interface buttons的一个常见集合。你明显地可以修改颜色方案来匹配你自己的网站。但是，每个按钮上悬停/激活(hover/active)状态真的给我深刻印象。虽然设计是”扁平”的，但你仍然注意到在你点击的按钮下有一个阴影。这真是漂亮的效果。能找到那里象这样的素材供免费使用，真是太好了。

简单按钮风格

这些简单按钮风格也是在codepen中找到的。这更多是个骨架式的例子，事实上没有支持一个激活状态。按钮悬停使用CSS3渐变，来以优雅的动画效果更新背景颜色。不要全信这个例子，看看你是否能扩展代码来适应你自己的需要。

Signup Form 注册表单

对于免费的HTML/CSS代码，CSS流片段(CSSFlow snippets)可能是最好的在线画廊之一。这个特别的例子是一个登陆表单 signup form，使用了扁平的按钮和一些输入域。你实际上可以试试 现场在线demo，看看它如何在你的浏览器里工作。提交按钮有自然地悬停和激活状态，加上当输入域在焦点时也发光。它不是那么可作为开始的东西，但是你能从CSSFlow snippet页面找到其他扁平化例子。

50个优美的扁平化设计网站

终极的扁平化设计指南怎么可能没有一些漂亮的例子呢？讨论和辩论扁平化设计很有意思，但是讨论不能总把你带到终点(however talking will not always get you to the finish line)。浏览其它布局的一些例子，确实能帮你把头脑中的界面组合在一起，这些可能你还没单靠自己去想过。
